# Завдання 5. Візуалізація обходу бінарного дерева

## Опис

У цьому завданні реалізовано візуалізацію обходу бінарного дерева двома способами:

- DFS — обхід у глибину;
- BFS — обхід у ширину.

Для кожного обходу вузли дерева отримують різні кольори. Колір змінюється від темнішого до світлішого залежно від порядку відвідування вузлів.

## Реалізація

За основу взято код із завдання 4 для побудови та відображення бінарного дерева: клас `Node`, функцію `add_edges`, функцію `draw_tree` та побудову дерева з масиву значень.

Функція `build_heap_tree` створює бінарне дерево зі списку значень. Для елемента з індексом `i`:

- лівий дочірній вузол знаходиться за індексом `2 * i + 1`;
- правий дочірній вузол знаходиться за індексом `2 * i + 2`.

DFS реалізовано у функції `dfs_traversal` за допомогою стеку.

BFS реалізовано у функції `bfs_traversal` за допомогою черги `deque`.

Після обходу функція `set_traversal_colors` задає кожному вузлу колір у форматі HEX. Перші відвідані вузли мають темніший колір, а наступні — світліший.

## Залежності

Для запуску цього завдання потрібні бібліотеки `matplotlib` та `networkx`. Їх можна встановити з файлу `requirements.txt`.

## Результат

Після запуску програма виводить у консоль порядок обходу DFS і BFS та відкриває вікна з візуалізацією дерева для кожного обходу.

Основний файл рішення: `main.py`.
