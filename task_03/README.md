# Завдання 3. Алгоритм Дейкстри

## Опис

У цьому завданні реалізовано алгоритм Дейкстри для пошуку найкоротших шляхів у зваженому графі.

Програма створює граф, задає ваги ребер і знаходить найкоротші відстані від початкової вершини до всіх інших вершин.

## Реалізація

Граф представлено у вигляді словника, де для кожної вершини зберігаються її сусіди та вага ребра до кожного сусіда.

Для вибору вершини з найменшою поточною відстанню використано бінарну купу через модуль `heapq`.

Функція `dijkstra` спочатку задає для всіх вершин нескінченну відстань, а для початкової вершини — відстань `0`.

Після цього алгоритм поступово дістає з купи вершину з найменшою відстанню і перевіряє, чи можна через неї знайти коротший шлях до сусідніх вершин.

Функція `restore_path` використовується для відновлення самого шляху від початкової вершини до потрібної вершини.

## Результат

Після запуску програма показує найкоротші відстані від вершини `A` до всіх інших вершин графа та відповідні шляхи.

Основний файл рішення: `main.py`.
