# Завдання 2. Дерево Піфагора

## Опис

У цьому завданні реалізовано побудову фрактала "дерево Піфагора" за допомогою рекурсії.

Програма дозволяє користувачу вказати рівень рекурсії, після чого будує дерево за допомогою модуля `turtle`.

## Реалізація

Для побудови дерева використано рекурсивну функцію `draw_pythagoras_tree`.

Базовий випадок рекурсії спрацьовує тоді, коли рівень рекурсії дорівнює нулю. У цьому випадку функція завершує роботу і не малює нові гілки.

У рекурсивному випадку програма малює одну гілку, після чого викликає цю саму функцію для лівої та правої частини дерева з меншим рівнем рекурсії.

Довжина кожної наступної гілки зменшується, тому дерево поступово формує фрактальну структуру.

## Результат

Після запуску програма просить ввести рівень рекурсії і відкриває вікно з побудованим фракталом.

Чим більший рівень рекурсії, тим детальнішим буде дерево. На більших рівнях, наприклад 9 або 10, побудова може займати більше часу, тому потрібно зачекати, поки програма завершить малювання. Але за цим процесом цікаво спостерігати.

Основний файл рішення: `main.py`.
