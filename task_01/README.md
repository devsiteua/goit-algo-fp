# Завдання 1. Однозв'язний список

## Опис

У цьому завданні реалізовано однозв'язний список та три операції з ним:

- реверсування списку через зміну посилань між вузлами;
- сортування списку методом вставок;
- об'єднання двох відсортованих списків в один відсортований список.

## Реалізація

За основу взято базову структуру однозв'язного списку з вузлами `Node`, де кожен вузол зберігає значення та посилання на наступний вузол.

Метод `reverse` проходить список і змінює напрям посилань між вузлами.

Для сортування використано алгоритм сортування вставками з часовою складністю O(n²).

Функція `merge_sorted_lists` послідовно порівнює елементи двох відсортованих списків і додає менший елемент у новий список.

## Результат

Після запуску програма демонструє:

- початковий список;
- список після реверсування;
- список після сортування;
- результат об'єднання двох відсортованих списків.

Основний файл рішення: `main.py`.
