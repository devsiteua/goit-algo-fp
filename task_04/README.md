# Завдання 4. Візуалізація бінарної купи

## Опис

У цьому завданні реалізовано візуалізацію бінарної купи.

Програма створює купу зі списку чисел, будує з неї бінарне дерево і відображає його за допомогою `networkx` та `matplotlib`.

## Реалізація

За основу взято код з умови завдання: використовується клас `Node`, функція `add_edges` для додавання вузлів і ребер у граф та функція `draw_tree` для відображення дерева.

Купа спочатку створюється зі списку чисел за допомогою `heapq.heapify`.

Після цього функція `build_heap_tree` будує дерево з масиву купи. Для кожного елемента з індексом `i`:

- лівий дочірній вузол знаходиться за індексом `2 * i + 1`;
- правий дочірній вузол знаходиться за індексом `2 * i + 2`.

Функція `visualize_heap` виводить початковий список, масив після побудови купи та відкриває вікно з візуалізацією бінарної купи.

## Залежності

Для запуску цього завдання потрібні бібліотеки `matplotlib` та `networkx`. Їх можна встановити з файлу `requirements.txt`.

## Результат

Після запуску програма показує початкові значення, масив купи та будує графічне представлення бінарної купи.

Основний файл рішення: `main.py`.
